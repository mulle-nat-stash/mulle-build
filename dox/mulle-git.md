## mulle-git usage

`usage: mulle-git [flags] <git options>`

Run git operations on the project and it's dependencies

```
mulle-git status -s
```


#### Flags

Flag        | Description                                   |
------------|-----------------------------------------------|
-n          | Dry run, don't actually execute               |
-v          | Verbose                                       |
-vv         | Very verbose                                  |
-vvv        | Extremely verbose                             |
-t          | Trace shell script                            |
