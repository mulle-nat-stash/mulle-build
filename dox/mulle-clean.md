## mulle-clean usage

Clean your project and it's dependencies

```
mulle-clean
```


#### Options

Option      | Description                                   |
------------|-----------------------------------------------|
-n          | Dry run, don't actually execute               |
-v          | Verbose                                       |
-vv         | Very verbose                                  |
-vvv        | Extremely verbose                             |
-t          | Trace shell script                            |
