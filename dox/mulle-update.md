## mulle-update usage

`usage: mulle-update [flags] [mulle-bootstrap options]`

Update (aka git pull) the project and it's dependencies

```
mulle-update
```


#### Flags

Flag        | Description                                   |
------------|-----------------------------------------------|
-n          | Dry run, don't actually execute               |
-v          | Verbose                                       |
-vv         | Very verbose                                  |
-vvv        | Extremely verbose                             |
-t          | Trace shell script                            |
