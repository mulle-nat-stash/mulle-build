## mulle-tag usage

`usage: mulle-clean [flags] <tag> [mulle bootstrap options]`

Tag your project and it's dependencies with a common tag

```
mulle-tag v2.1
```


#### Flags

Flag        | Description                                   |
------------|-----------------------------------------------|
-n          | Dry run, don't actually execute               |
-v          | Verbose                                       |
-vv         | Very verbose                                  |
-vvv        | Extremely verbose                             |
-t          | Trace shell script                            |
